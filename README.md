## Wellcome

해당 프로젝트는 기본적인 자료구조와 알고리즘을 구현한 프로젝트입니다.

## 필수 사항

- iOS12.2 +
- XCode10.2 +

## 인스톨 방법

SCDataStructure는 cocoapod [CocoaPods](https://cocoapods.org)으로 해당 라이브러리를 설치가 가능합니다.
Podfile에 아래의 코드를 추가하고 pod install 하시길 바랍니다.

```ruby
pod 'SCDataStructure'
```
